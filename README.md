# Emotion Detection Application

An AI-powered web application that performs emotion detection on customer feedback
using the Watson NLP Embed library. Built with Python, Flask, and deployed as a web app.

## Project Structure
- EmotionDetection/ — Core NLP package
- server.py — Flask web server
- test_emotion_detection.py — Unit tests
- templates/index.html — Frontend UI

## Features
- Detects: anger, disgust, fear, joy, sadness
- Returns dominant emotion
- Error handling for blank input
- Pylint score: 10.00/10
